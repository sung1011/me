## 相对论

### 时间膨胀

```bash

   地球沿某方向运动, 从地球后面追过来的光 与 地球前面过来的光 速度一样, 
   虽然从理解上追光速度更快, 实际上速度相同(光速)

    追赶的光------------------->  地球>  <------迎面的光
  (速度快, 时间慢)

    因为 速度 = 路程 / 时间, 速度固定(光速), 路程长的(追赶的光), 时间更短
    所以 `速度越快, 时间越慢`, 并推倒出 `质量/重力越大, 时间越慢`, `速度越快, 长度越短`

# 两个人a,b都是20岁, a坐飞船达到光速, a一年后回到地球, a=21岁, b=90岁

# 大星球时间慢 如: 黑洞/宇宙中心/恒星  ---- 地球139亿年, 宇宙中心只过了1~2秒

# 小星球时间快 如: 火星, 量子世界 ---- 高等文明更可能出现在这些发展时间久(时间快)的地方

# 假说
#   1. 光速在观测时坍缩成了光速
#   2. 世界是虚拟的, 只有光和重力是真实世界传过来的
```

### 质能等价

```bash
# 质能方程
    E=mc² 能量和质量是等价的并且可以互换

# 质量 => 能量
    核裂变过程
        向`铀-235`中打入一个`中子`, 试其分裂成2~3个`更轻的原子核` + 2~3自由中子 + 一些中微子, 并释放巨量能量

                    ooo          oo       oo
        o    +      ooooo  ==>   ooo   +  ooo  + x + x + x
                    ooo          oo       oo

        中子          铀-235       原子  +  原子 + 2~3个自由中子

    `更轻的原子核` 变轻的原因就是: 质量转化为了能量

    # 转化比例 
        根据公式mc², 1g物质 = 90兆焦耳能量 (参照: 广岛原子弹释放54兆焦耳)

# 能量 => 质量
    电子加速
        用电磁场 在环形轨道上 加速一个电子
        加速到光速后, 继续注入能量
        结果速度不变(光速), 质量变重了
```
