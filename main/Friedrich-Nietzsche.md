# 尼采

## 生平

```bash
    1844.10.15 ~ 1900.8.25,
        早年（1844—1869）
            出身德国的古典语言学家和哲学家
            大学研究 神学 和 古典语文学, 之后失去对基督教的信仰, 开始专攻哲学.
        教授生涯（1869—1879）
            成为教授, 短暂军旅生涯, 发表论文和演讲, 健康变差(视力近乎丧失, 偏头疼, 剧烈胃疾), 并且车祸和大病了一场.
        孤独的哲学家（1879—1888）
            孤独的写书, 自费出书, 病情加重
        发疯和去世（1889—1900）
            发疯, 死于肺炎, 书大卖特卖

# 发疯 广场看见一匹马被马夫鞭打，突然上前抱住马的脖子痛哭道：“我受苦受难的兄弟啊！”，接着便瘫倒在地上

# 出书 出了4本<查拉图斯特拉如是说>, 之后又出了多本书, 对这4本进行注解
```

## 成就

```bash
    `上帝已死` 

        世界已经不存在绝对的真理(即 上帝)
        如果认可上帝, 最终会走向虚无主义, 成为行尸走肉的"末人"
        放弃对上帝的信仰(承认上帝已死), 去重新认识世界, 成为崭新未来的"超人"

        # `社会环境` 地动说, 进化论动摇了

        # `虚无主义` 上帝建立在主观观念上, 不是客观普世的道德法律, 最终会走向虚无主义

        # `超人` 骆驼阶段: 忍耐承受压力苦难; 狮子阶段: 挣脱束缚做想做的事情; 儿童阶段: 向儿童一样探索追求梦想

    `永恒轮回`

        每次死亡人生都会重新来过, 每一世过得都一样, 无限循环重复, 不论做什么同样的苦难都会发生.
        命运不能改变, 只能改变自己的看法, 接受痛苦, 享受痛苦.

        # 幸福 从痛苦中解脱出来, 获得永恒的幸福.

        # 烈性药 挣扎无用, 在苦海中游泳, 只有经历过绝望的人能读懂尼采

        # 日记 (虽然我的人生充满苦难)"我怎么会不爱我的人生呢, 正因为我很爱它, 所以我才不断的对我自己讲述我的人生"

    `主人-奴隶道德说`

        主人道德的主要特征是：自我肯定、自豪、主动。 标准为好/坏
        奴隶道德则是：自我否定、谦卑、被动、怜悯。 标准为善/恶
        一个人是被哪种道德主宰不是由他的身份地位决定的，而是由他的行为中蕴含的心态。
        一个大独裁者也可能属于奴隶道德控制，因为他的所作所为可能是由怨恨与报复心所推动。

        这种区分方式直接指出“好与坏”与“善与恶”两种道德标准的冲突，
        更重要的是，在主人道德中的“好”却在奴隶道德中被视为是“恶”。

```
