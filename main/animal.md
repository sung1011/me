## 恐龙

```bash
    存在
        2亿5000w年前 - 6500w年前

        # 进化论 无法解释恐龙进化这么慢, 人类为什么进化这么快
    灭绝
        6500w年前, 小行星撞击墨西哥 200km/s; 威力 = 广岛原子弹的10亿倍 = 通古斯大爆炸100w倍
        引发大地震, 地震引发海啸, 席卷世界
        冲击波将砂石和海水掀到大气层, 墨西哥岩层中含有大量的石膏, 石膏燃烧变成硫, 导致天空永远被黑暗笼罩, 下着酸雨, 持续30年
        阳光照射不进来, 导致植物死亡, 动物死亡, 最终物种灭绝 (第五次生物大灭绝)

    骨骼
        以化石推算, 恐龙的骨骼应该无法支撑身体, 翼龙的体型也无法飞行
        可能是当时地球的重力没有那么大
        后来重力变大导致恐龙灭绝
        # 自转变慢导致离心力下降, 导致重力变大
        #  假说1 小行星撞击地球导致自转变慢 (可能性不大)
        #  假说2 (从天而降的)大洪水导致地球质量变大, 自转变慢
```
