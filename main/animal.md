## 恐龙

```bash
    存在
        2亿5000w年前 - 6500w年前

        # 进化论 无法解释恐龙进化这么慢, 人类为什么进化这么快
    灭绝
        6500w年前, 小行星撞击墨西哥 200km/s; 威力 = 广岛原子弹的10亿倍 = 通古斯大爆炸100w倍
        引发大地震, 地震引发海啸, 席卷世界
        冲击波将砂石和海水掀到大气层, 墨西哥岩层中含有大量的石膏, 石膏燃烧变成硫, 导致天空永远被黑暗笼罩, 下着酸雨, 持续30年
        阳光照射不进来, 导致植物死亡, 动物死亡, 最终物种灭绝 (第五次生物大灭绝)

    骨骼
        以化石推算, 恐龙的骨骼应该无法支撑身体, 翼龙的体型也无法飞行
        可能是当时地球的重力没有那么大
        后来重力变大导致恐龙灭绝
        # 自转变慢导致离心力下降, 导致重力变大
        #  假说1 小行星撞击地球导致自转变慢 (可能性不大)
        #  假说2 (从天而降的)大洪水导致地球质量变大, 自转变慢
```

## 长颈鹿

```bash
    属性
        身高 6m
        体重 2000kg
        速度 50km/h
    攻击
        前踢
        后踢
        头槌 头上有5个角, 依靠脖子的甩动攻击, 一般攻击同类时使用
    睡眠
        20min/day 食草动物中最短
    不符合进化论
        长颈鹿 需要同时协调完成进化.
            大脑 "迷网"组织, 控制血压 (需要优先进化)
            心脏 变强
            脖子 变长
            四肢 变长
        不符合进化论 (一般只单一进化, 也不可能优先进化大脑)
        符合病毒进化论 (可以系统性变异)
```
