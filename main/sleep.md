# sleep

## 时长

| 物种     | 时长   | 例子                |
| -------- | ------ | ------------------- |
| 人类     | 8h     |
| 食草动物 | 2h     | 长颈鹿 20min (最短) |
| 食肉动物 | 10~20h |
| 植物     | 0h     |

## 犯困

- 肉体 吃食物
- 精神 吃信息

## 睡眠周期

- 快速眼动 做梦有记忆

- 非快速眼动 做梦无记忆

## 睡眠的记忆加深

- 身体记忆
  - 吉他
  - 自行车
  
- 情感记忆
  - 开心快乐
  - 恐惧
  - 失恋创伤
  - 身体受伤
  - 哭着睡觉

> `不会加深的记忆` 对生命没有意义的事情, 如: 背单词, 数学公式, 死记硬背
