# PCA principal component analysis  主成分分析

主成分分析（Principal Component Analysis, PCA）是一种常用的无监督学习方法，用于降维和数据可视化。PCA通过线性变换将数据投影到低维空间，保留最大方差的方向作为主成分。

## 原理

```js
// 1 原始数据
            |              .
            |           .  .
            |      . .  .
            |     ..
            |
            |
--------------------------------
            |
            |
            |
            |
            |
            |

// 2 坐标系移动到数据中心
       |   .
       |.  .
---.-.-|.-----------
  ..   |  
       |   

// 3 旋转坐标系 使得数据方差最大(数据在x轴的投影最不重叠)
\     . /
 \    ./
  \  ./
   \ /.
    X
   . \
  /.  \
 ..    \
/       \

// 4 保留主成分 只保留在x轴的投影
       |    
       |    
---....|-.-...------
       |  
       |   

// 5 得到一维数据
[-4, -3, -2, -1, 2, 4, 5, 6]
```

## 意义

- 数据降维
- 数据可视化
- 噪声过滤

## 缺点

- 线性变换: 无法处理非线性关系
- 方差最大化: 保留了数据中的主要信息，但可能丢失一些细节

## 实例

- 人脸识别
- 图像压缩
- 金融数据分析
