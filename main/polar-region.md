## 南极

```bash
# 温度
    -50° (历史最低温度2010年 -93.2°)
    海水温度低, 含氧量高
    # 海拔
        2000m (所以冰层很厚至少也2000m)
    # 洋流
        南极环流 周围的暖流进不来
    # 冰
        90%世界上的冰, 且是淡水冰, 即70%淡水在南极
# 生态
    # 植物
        无
    # 动物
        企鹅
        鲸鱼
        海豹
# 国家
    不属于任何国家, 属于全人类
# 极昼极夜
    前半年极夜, 后半年极昼
# 人类居住
    无
# 发现
    # 化石
        恐龙, 热带植物, 大量陨石
    # 金字塔
    # 巨型生物
        巨型海蜘蛛, 巨型海鳞虫, 大虾, 大王乌贼, 迷相生海月(水母, 40m, 世上最长生物)
    # 冰下湖
        # 勘探
            凸起冰峰的下面是山脉
            平坦冰地的下面是湖泊
        # 挖掘
            冰层很厚(>2000m), 湖泊古老(3000万年前)且极冷, 可能存在巨大远古生物,
            钻头可能污染湖泊, 破坏生态 
```

## 含氧量

```bash
    (海水, 空气)温度越低 -> 含氧量高 -> 生物巨大


# 古代空气含氧量都高 导致生物都巨大; 恐龙, 猛犸, 蜻蜓(1~2m), 蟑螂(0.5m)

# 当水含氧量足够高, 人类肺泡可以吸入高氧水, 代替呼吸
# 当海水含氧量很高, 人类可以潜入更深的海底 (肺中的空气会被压力压塌, 但肺中的高氧水则不会)

# 
```
